## Files

- `add.c` - Takes two inetgers as input and prints their sum.
- `agree.c` - Asks question from the user and prints the decision.
- `cash.c` - Takes input of money to return and tells minimum total number of coins required to return the money in value of 1, 5, 10, and 25 cents using Greedy algorithm.
- `cat.c` - Prints meow three times.
- `compare.c` - Compares two integers as taken from input by the user.
- `contacts.c` - Creates a contact.
- `credit.c` - Checks the authenticity of a card and displays its type using Luhn's algorithm
- `hello.c` - Greets the user.
- `helloworld.c` - Basic hello world programme.
- `mario-less.c` - Prints a right aligned pyramid of a height 1 to 8 as given to it by the user.
- `mario-more.c` - Pritns two pyramids of height 1 to 8 as given by the user.
- `my-cat.c` - Prints meow as many times the user asks, by providing input.
- `n-mewo.c` - Prints meow as many times as the user wants, but declares a function for it.
- `pillar.c` - Prints a pillar of #, of dimensions given by the user.
- `pyramid.c` - Prints a left aligned pyramid of #, of height ranging from 1 to 8.
- `scrabble.c` - A two player Scrabble game.
- `sum-cons-natural-nos.c` - Prints the sum of consecutive natural numbers as per the given input.
- `sum-multiple-int.c` - Sums as many numbers the user wants.
- `wall.c` - Prints a square wall of # as per the dimesions asked by the user.
